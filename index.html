🤖 Discord Gmail Bot

📧 Manage Gmail right from Discord!

Gmail Bot is a powerful Discord bot that lets you:

  • Connect and authenticate your Gmail account
  • Read unread emails
  • View full email content by index
  • Delete unread emails, ads, or even entire inbox
  • Add custom labels to emails
  • Check and empty your trash folder
All actions are done inside Discord, securely and easily.
No need to switch tabs or share your credentials.

🔐 Security & Privacy

  • OAuth 2.0 – you authorize Gmail access yourself
  • No passwords are stored
  • Access tokens are encrypted and per-user
  • You can revoke access: https://myaccount.google.com/permissions

⚙️ How to Use

  1. Run /connect_gmail
  2. Follow the auth link from the bot
  3. Get the code and run /auth <code>
  4. Done! Use commands like /read_unread, /del_ads, etc.
